# Historical Dates — запуск

## Требования

- **Node.js**: желательно `18+` (или `20+`)
- **npm**

---

## Локальный запуск

### 1) Установка зависимостей

```bash
npm install
```

### 2) Запуск dev-сервера

```bash
npm run dev
```

Открой адрес из консоли (обычно `http://localhost:3000`).

---

## Сборка проекта

### 1) Build

```bash
npm run build
```

### 2) Локальный просмотр production-сборки

```bash
npm run preview
```
